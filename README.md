# Business Sales Performance Analytics

This project analyzes sales performance using the Superstore dataset and Tableau.

## Objectives
- Analyze revenue trends
- Identify top-performing products
- Evaluate category profitability
- Assess regional sales performance

## Tools Used
- Tableau
- Superstore Sales Dataset

## Key Insights
- Technology category generates the highest revenue
- East region shows the highest profit
- Sales show an overall upward trend over time

## Deliverables
- Interactive Tableau Dashboard
- Sales performance insights
- Business recommendations
